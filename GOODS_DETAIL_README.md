# 商品详情功能说明

## 功能概述

点击商品卡片后，会在新标签页中打开独立的商品详情页面，展示完整的商品信息和数据分析。

## 页面路径

- **详情页面**：`frontend/goods_detail.html`
- **URL格式**：`goods_detail.html?id={product_id}`

## 功能特性

### 1. 顶部导航
- 返回按钮（返回上一页或首页）
- 页面标题
- 收藏按钮（待实现）
- 分享按钮（待实现）

### 2. 商品基本信息
- **左侧**：商品主图（大图展示，固定定位）
- **右侧**：
  - 商品标题
  - 商品标签（多个标签展示）
  - 价格信息（现价、原价、优惠券）
  - 佣金信息（佣金比例、预估佣金）
  - 店铺信息（店铺logo、店铺名称、店铺ID）
  - 销售数据（总销量、浏览量、带货达人数）
  - 操作按钮（查看原商品、下载）

### 3. 数据分析标签页

#### 基础分析
- **核心数据**：6个关键指标卡片
  - 浏览量
  - 商品销量
  - 商品销售额
  - 支付买人
  - 支付直播
  - 支付视频
- **趋势图表**：
  - 商品销量趋势
  - 商品销售额&浏览量

#### 达人分析
- 每日大获取视频&总次数
- 新增大获取人&总次数

#### 视频分析
- 视频数据分析（待开发）

#### 直播分析
- 直播数据分析（待开发）

## 使用方法

### 从首页打开
1. 在首页商品列表中，点击任意商品卡片
2. 自动在新标签页中打开商品详情页面

### 直接访问
访问 URL：`goods_detail.html?id={product_id}`

例如：`goods_detail.html?id=3797210999149429067`

## 技术实现

### 前端
- 使用 Vue 3 响应式数据
- Tailwind CSS 样式
- Material Symbols 图标
- 响应式布局（支持移动端）
- URL参数传递商品ID

### 后端API
- `GET /api/goods/detail/{product_id}` - 获取商品详情

### 数据来源
- 基本信息：从商品详情API获取
- 分析数据：目前为占位数据（显示为0或"暂无数据"）

## 页面状态

### 加载状态
- 显示加载动画
- 提示"加载中..."

### 错误状态
- 显示错误图标和错误信息
- 提供返回按钮

### 正常状态
- 显示完整的商品信息和数据分析
